
# uuid-base64

Compress a UUID to 22 characters

```js
var base64 = require('uuid-base64');
base64('0e8c72e5-463f-446e-bcf2-695274d55940');
// => Doxy5UY_RG688mlSdNVZQA
```

## Installation

  node:

```
$ npm install uuid-base64
```
